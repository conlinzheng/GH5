# GH5 项目高度自定义实现 - 产品需求文档

## Overview
- **Summary**: 对 GH5 项目进行梳理，实现页面显示元素、图片、文字、译文等的高度自定义，确保产品系列图片、轮播图和图标等资源正确引用。
- **Purpose**: 使项目具有高度的可定制性，方便用户根据需要修改页面内容和资源。
- **Target Users**: 项目维护者和内容管理者。

## Goals
- 实现页面显示元素、图片、文字、译文的高度自定义
- 确保产品系列图片以产品图文件夹为准，正确识别主图和详情图
- 确保轮播图以轮播图图片目录为准
- 确保图标以页面临时图标文件夹内应用
- 删除与项目无关的文件

## Non-Goals (Out of Scope)
- 不修改项目的基本架构和技术栈
- 不添加新的功能模块
- 不改变现有的路由结构

## Background & Context
- GH5 项目是一个基于 Vue 3 的产品展示网站
- 项目当前使用硬编码的产品数据和图片路径
- 项目已部署到 Vercel，地址为 https://gh-5-chi.vercel.app/
- 项目包含产品图和轮播图等资源目录

## Functional Requirements
- **FR-1**: 实现产品数据的动态加载，从产品图文件夹中读取产品信息
- **FR-2**: 实现图片的正确识别和引用，区分主图和详情图
- **FR-3**: 实现轮播图的动态加载，从轮播图目录中读取图片
- **FR-4**: 实现图标的动态加载，从页面临时图标文件夹中读取
- **FR-5**: 实现文字和译文的可配置化

## Non-Functional Requirements
- **NFR-1**: 代码结构清晰，易于维护
- **NFR-2**: 加载性能良好，资源加载速度快
- **NFR-3**: 配置文件结构合理，易于修改
- **NFR-4**: 错误处理机制完善，确保页面正常运行

## Constraints
- **Technical**: 基于现有 Vue 3 + Vite 技术栈
- **Business**: 保持项目的基本功能不变
- **Dependencies**: 依赖本地文件系统读取资源

## Assumptions
- 产品图文件夹结构保持不变
- 轮播图文件夹结构保持不变
- 图标文件夹结构保持不变

## Acceptance Criteria

### AC-1: 产品数据动态加载
- **Given**: 产品图文件夹中存在产品数据
- **When**: 页面加载时
- **Then**: 系统自动读取产品图文件夹中的产品信息并显示
- **Verification**: `programmatic`
- **Notes**: 产品数据应从产品图文件夹中的 products.json 文件读取

### AC-2: 图片正确识别和引用
- **Given**: 产品图文件夹中存在图片文件
- **When**: 页面加载时
- **Then**: 系统自动识别主图（带(1)的图片）和详情图，并正确引用
- **Verification**: `programmatic`
- **Notes**: 图片名称带(1)的为主图，其他数字的为详情图

### AC-3: 轮播图动态加载
- **Given**: 轮播图文件夹中存在图片文件
- **When**: 页面加载时
- **Then**: 系统自动读取轮播图文件夹中的图片并显示
- **Verification**: `programmatic`
- **Notes**: 轮播图应从轮播图文件夹中读取

### AC-4: 图标动态加载
- **Given**: 页面临时图标文件夹中存在图标文件
- **When**: 页面加载时
- **Then**: 系统自动读取页面临时图标文件夹中的图标并应用
- **Verification**: `programmatic`
- **Notes**: 图标应从页面临时图标文件夹中读取

### AC-5: 文字和译文可配置化
- **Given**: 配置文件中存在文字和译文配置
- **When**: 页面加载时
- **Then**: 系统根据配置文件显示文字和译文
- **Verification**: `programmatic`
- **Notes**: 文字和译文应从配置文件中读取
